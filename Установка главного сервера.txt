Установка главного сервера:

1) Открыть порты 23563,23564 TCP/UDP
2) Установить .NET Framework 4.6.1,  Visual Studio 2017 Community в папку "C:\Program Files (x86)\Microsoft Visual Studio" и авторизироваться в ней
3) Зайти в папку Client
4) Открыть с помощью блокнота файл Client.csproj
5) Найти фразу "ПУТЬ ТУТ"
6) Заменить эту фразу путем к папке, где находится приложение MainServer.exe
7) Сохранить
8) Открыть в папке Client файл Client.sln
9) Удалить ссылку "Omdageraaa.dll"
10) Добавить ссылку на "Omdageraaa.dll", который находится в папке с MainServer.exe
11) Скомпилироваться проект и закрыть его
12) Залить файл "DJinglw.dll" на хостинг
13) Скопировать прямую ссылку на этот файл
14) Вставить эту ссылку в файл ModuleLink.txt и нажать сохранить файл
15) Вписать в ip.cfg айпи дедика, на котором главный сервер
16) Запустить от администратора приложение MainServer.exe

Подключение старой панели к главному серверу 

1) Закрыть панель ботнета
2) В 2ip.cfg прописать айпи дедика с главным сервером
3) Запустить панель
4) Зарегистрироваться
5) Активировать себя путем:
   а) Зайти на гланый сервер
   б) Нажать кнопку Users
   в) Найти свой логин
   г) Нажать правой клавишей мыши на свой логин
   д) Выбрать в контекстном меню "Активировать"
6) Авторизоваться в панели
